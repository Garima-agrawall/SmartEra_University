# 🎓 SmartEra University – Full Stack Web Application

SmartEra University is a full-stack web application designed to manage and automate university operations efficiently. It provides a centralized system to handle student data, courses, faculty, and administrative tasks.

---

## 🚀 Features

- Student Registration & Login  
- Course Management System  
- Faculty Management  
- Admin Dashboard  
- Secure Authentication  
- Campus AI Chatbot (AI-based query system)  
- Contact Form with Database Integration  
- Responsive and User-Friendly UI  

---

## 🤖 Campus AI Chatbot

Campus AI is an intelligent chatbot integrated into the system that:

- Answers queries about:
  - Fees  
  - Admissions  
  - Courses  
  - Campus information  
- Uses Google Gemini API  
- Provides short and relevant responses  
- Restricts answers to university-related queries
